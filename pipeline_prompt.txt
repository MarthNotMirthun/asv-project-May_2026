─────────────────────────────────────────────────────────
STAGE 1A — Use hw-validation:
─────────────────────────────────────────────────────────
"Read CLAUDE.md. Review [TARGET] for electrical hardware
interface correctness.

Use Puppeteer to navigate directly to manufacturer datasheet
PDFs for pin diagrams, timing waveforms, and voltage tables.
Use web search for text specs and known failure modes.
Cite every spec with exact source URL and page/section.

Check:
- Every port connecting to real hardware: voltage levels,
  timing, current requirements
- Every .cst pin: exists on GW2AR-18, LVCMOS33-compatible
  bank, supports required frequency
- AD9226 DVDD vs Tang Nano GPIO voltage compatibility
- JSN-SR04T echo pin voltage vs ESP32 GPIO max input
- Power budget: buck converter headroom >20%
- CLKS_PER_BIT=234 for 115200 baud at 27MHz
- adc_clk output port drives AD9226 ENCODE pin
- Clock domain crossings have 2-FF synchronizers

At the end of your output include CROSS-CHECK FLAGS:
Any finding with DSP or systems implications for the
other validators."

Collect complete hw-validation output.

─────────────────────────────────────────────────────────
STAGE 1B — Use dsp-signal-validator:
─────────────────────────────────────────────────────────
"Read CLAUDE.md. Review [TARGET] for DSP mathematical
and signal chain correctness.

Use web search and Puppeteer to verify specs from
authoritative DSP references and manufacturer datasheets.
Cite every spec with source.

Check:
- Number format at every stage boundary:
  adc_interface→cic_decimator→fir_filter→matched_filter→
  peak_detector→uart_tx
- AD9226 offset binary→two's complement conversion exists
  ({~data[11], data[10:0]}) — absence is BLOCKER
- CIC internal width = 34 bits minimum (12 + ceil(3×log2(160)))
- CIC output right-shift = 5 bits for 16-bit output precision
- All accumulators: saturation not wrapping overflow
- OTR flag propagates as sample_otr through every stage
- Every valid signal delayed to match data pipeline latency
- Q1.15 multiply: intermediate Q2.30, shift right 15

At the end include CROSS-CHECK FLAGS:
Any finding with electrical or systems implications."

Collect complete dsp-signal-validator output.
