\# ASV Acoustic Homing USV



GPS-denied autonomous catamaran that homes on two floating 

acoustic beacons using FPGA matched filter signal processing.



\*\*Demo deadline:\*\* August 10, 2026  

\*\*Stack:\*\* Tang Nano 20K FPGA · Raspberry Pi 4 · ROS 2 Jazzy · ESP32 micro-ROS  

\*\*Signal processing:\*\* Dual LFM chirp matched filters (34–38kHz / 42–46kHz)  

\*\*Navigation:\*\* Dead reckoning EKF + acoustic range correction  



Built as a portfolio project targeting naval defense roles 

in embedded systems and signals.
